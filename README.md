## Design Direction

For my portfolio, I pursued an **old art book** aesthetic.

- **Header Font:** Dawning of a New Day
- **Body Font:** Cormorant Garamond
- **Primary Color:** #510D18
- **Accent Color:** #93B385 #E4CC6D
- **Background:** #FFFFFF
